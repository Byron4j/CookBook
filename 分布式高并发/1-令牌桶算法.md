# 令牌桶算法


Google的Guava实现了一个令牌同算法类：```com.google.common.util.concurrent.RateLimiter```.
引入依赖可以查看该类的源码：
```xml
<dependency>
    <groupId>com.google.guava</groupId>
    <artifactId>guava</artifactId>
    <version>20.0</version>
</dependency>
```
