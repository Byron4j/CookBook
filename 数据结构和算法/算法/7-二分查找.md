# 二分查找算法

## 1. 概念

二分查找（Binary Search）是一种在有序数组中查找目标元素的高效算法。

- 时间复杂度：`O(log n)`
- 空间复杂度：`O(1)`（迭代版）或 `O(log n)`（递归版）
- 前提条件：**数组必须有序**
- 每次将搜索范围缩小一半

## 2. 来源

二分查找是最经典的查找算法之一，思想简单但极其高效，广泛应用于各种场景。

## 3. 知识点

### 3.1 核心思想

```
二分查找过程:

数组: [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]
查找: 23

Step 1:
  left = 0, right = 9
  mid = (0 + 9) / 2 = 4
  arr[4] = 16
  16 < 23 → 在右半部分
  left = mid + 1 = 5

Step 2:
  left = 5, right = 9
  mid = (5 + 9) / 2 = 7
  arr[7] = 56
  56 > 23 → 在左半部分
  right = mid - 1 = 6

Step 3:
  left = 5, right = 6
  mid = (5 + 6) / 2 = 5
  arr[5] = 23
  找到！索引 = 5

查找次数: 3 次（数组大小 10）
对比线性查找: 最多需要 6 次
```

### 3.2 边界问题

```
边界处理（避免死循环）:

标准写法（左闭右闭区间）:
left = 0, right = n - 1
while (left <= right):
    mid = left + (right - left) / 2  // 避免溢出
    if (arr[mid] == target) return mid
    else if (arr[mid] < target) left = mid + 1
    else right = mid - 1
return -1

为什么 mid = left + (right - left) / 2 ?
避免 (left + right) / 2 溢出
当 left 和 right 很大时，left + right 可能溢出 int

为什么 left <= right ?
确保当 left == right 时仍检查该元素
如果写成 left < right，会漏掉最后一个元素
```

### 3.3 查找边界

```
查找第一个等于 target 的元素:

left = 0, right = n - 1
result = -1
while (left <= right):
    mid = left + (right - left) / 2
    if (arr[mid] == target):
        result = mid       // 记录结果
        right = mid - 1    // 继续在左半部分查找
    else if (arr[mid] < target):
        left = mid + 1
    else:
        right = mid - 1
return result

示例: [1, 2, 2, 2, 3], target = 2
查找第一个 2: 索引 1


查找最后一个等于 target 的元素:

类似，但找到后 left = mid + 1

示例: [1, 2, 2, 2, 3], target = 2
查找最后一个 2: 索引 3


查找第一个大于等于 target 的元素（lower_bound）:

left = 0, right = n  // 注意 right = n
while (left < right):
    mid = left + (right - left) / 2
    if (arr[mid] < target):
        left = mid + 1
    else:
        right = mid
return left

示例: [1, 3, 5, 7], target = 4
返回索引 2（元素 5）


查找第一个大于 target 的元素（upper_bound）:

left = 0, right = n
while (left < right):
    mid = left + (right - left) / 2
    if (arr[mid] <= target):
        left = mid + 1
    else:
        right = mid
return left

示例: [1, 3, 5, 7], target = 5
返回索引 3（元素 7）
```

## 4. 数据结构

二分查找直接在有序数组上操作：

- 输入：`int[]`（必须有序）
- 无需额外空间

```java
public class BinarySearch {
    // 基本二分查找
    public static int search(int[] arr, int target) {
        int left = 0, right = arr.length - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            
            if (arr[mid] == target) {
                return mid;
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return -1;  // 未找到
    }
    
    // 查找第一个等于 target 的索引
    public static int firstIndex(int[] arr, int target) {
        int left = 0, right = arr.length - 1;
        int result = -1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            
            if (arr[mid] == target) {
                result = mid;
                right = mid - 1;  // 继续在左半部分查找
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return result;
    }
    
    // lower_bound: 第一个大于等于 target 的索引
    public static int lowerBound(int[] arr, int target) {
        int left = 0, right = arr.length;
        
        while (left < right) {
            int mid = left + (right - left) / 2;
            
            if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        
        return left;
    }
}
```

## 5. 最佳实践

1. **确保数组有序**后再使用二分查找。  
2. **注意边界条件**，避免死循环和越界。  
3. **优先使用迭代版**，避免递归栈溢出。  
4. **查找边界时使用 left < right**。  
5. **中点计算用 `left + (right - left) / 2`** 防止溢出。  

### 应用场景

```
经典应用:

1. 查找元素是否存在
   int index = binarySearch(arr, target);

2. 查找插入位置
   int pos = lowerBound(arr, target);

3. 统计元素出现次数
   int first = firstIndex(arr, target);
   int last = lastIndex(arr, target);
   int count = last - first + 1;

4. 查找最接近的元素
   int pos = lowerBound(arr, target);
   if (pos == 0) return arr[0];
   if (pos == n) return arr[n-1];
   return Math.abs(arr[pos]-target) < Math.abs(arr[pos-1]-target) 
          ? arr[pos] : arr[pos-1];

5. 旋转数组查找
   [4, 5, 6, 7, 0, 1, 2] 查找 0
   需要修改二分查找逻辑

6. 二分答案
   在答案空间中二分，如寻找满足条件的最小值
```
